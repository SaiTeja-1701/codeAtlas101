
# Onboarding Instructions
To get started with the project, follow these steps:
1. **Clone the repository**: Clone the repository using `git clone`
2. **Install dependencies**: Install the dependencies using `npm install`
3. **Start the server**: Start the server using `npm start`
4. **Access the application**: Access the application at `http://localhost:3000`

## Setup Instructions
To set up the project, follow these steps:
1. **Create a new MongoDB database**: Create a new MongoDB database for the application
2. **Update the database connection string**: Update the database connection string in the `server.js` file
3. **Start the server**: Start the server using `npm start`

### Contribution Guidelines
To contribute to the project, follow these guidelines:
1. **Fork the repository**: Fork the repository using `git fork`
2. **Create a new branch**: Create a new branch for your changes
3. **Make changes**: Make changes to the code and commit them using `git commit`
4. **Create a pull request**: Create a pull request to merge your changes into the main branch
